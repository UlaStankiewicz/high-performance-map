# High Performance Map -coding challenge
Application for adding points map and then sorting results


## Tools
FRONTEND: React, Create React app,  Typescript, scss, axios, react-hook-google-maps
BACKEND: NodeJS, MongoDB, Typescript, Express


## Installation
## Frontend
1. Clone the repo: `https://github.com/sudolann/high-performance-map.git`
2. `cd high-performance-map/frontend`
3. Install packages: `npm install`
4. Build project and launch: `npm start`
5. Open your browser at: `http://localhost:3000`

## Backend
1. cd `cd high-performance-map/frontend`
2. npm install
3. npm start
